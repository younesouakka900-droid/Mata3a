الإعداد الأولي — يظهر مرة واحدة فقط
قبل البدء:
1. اذهب إلى supabase.com → أنشئ مشروعاً مجانياً
2. في SQL Editor نفّذ:
create table if not exists media (id uuid default gen_random_uuid() primary key, name text not null, type text not null, url text not null, created_at timestamptz default now());
3. في Storage → Buckets → تحقق من اسم الـ Bucket
4. من Project Settings → API انسخ URL و anon key
